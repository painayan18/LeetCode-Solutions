# Reverse Linked List
Given the head of a singly linked list, reverse the list, and return the reversed list.
<br/><br/>

#### Example 1:

Input: `head = [1,2,3,4,5]`<br/>
Output: `[5,4,3,2,1]`<br/>
<br/>

#### Example 2:

Input: `head = [1,2]`<br/>
Output: `[2,1]`<br/>
<br/>

#### Example 3:

Input: `head = []`<br/>
Output: `[]`<br/>
<br/>

#### Constraints:

- The number of nodes in the list is the range `[0, 5000]`.
- `-5000 <= Node.val <= 5000`
